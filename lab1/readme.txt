Приложение, позволяющее пользователю выбирать, а затем интерактивно менять цвет,
показывая при этом его составляющие в трех моделях одновременно CMYK – RGB – HSV.


Для каждой компоненты каждой из трех цветовых моделей создается переменная tkinter.IntVar.
Для каждой такой переменной создается поле для ввода и слайдер. При изменении значения
любой цветовой компоненты одной из моделей происходит вызов соответствующей функции для пересчета значений
цветовых компонент для двух других моделей.

Если в поле для ввода ввести отрицательное число оно будет заменено нулем.
Если ввести число, превышающее максимальное для данной компоненты, оно будет заменено
максимальным допустимым.

Выбранный цвет отображается в окне снизу, цвет которого задается значениями компонент модели RGB.

По нажатию кнопки 'Choose Color' открывается цветовая палитра, в которой можно задать цвет
в модели RGB. После чего значения в других цветовых моделях обновятся автоматически.
