Вариант 7
Программа демонстрирует работу низкочастотных фильтров(гаусса, медианный) и 2 метода глобальной пороговой обработки (Оцу, Бернсена(глобальная версия)).

Области применения метода Бернсена:

	1.	Оптическое распознавание символов (OCR): Используется для улучшения качества изображений текста.
	2.	Медицинская визуализация: Применяется для обработки рентгеновских снимков, УЗИ и других медицинских изображений.
	3.	Анализ микроскопических изображений: В биологии и химии метод помогает выделить микрообъекты, такие как клетки и бактерии, на сложных фонах.

Области применения метода Оцу:

	1.	Обработка документов: Применяется для бинаризации отсканированных документов, где текст должен быть четко отделен от фона.
	2.	Компьютерное зрение: Используется для выделения объектов в кадре.
	4.	Медицинские изображения: Метод Оцу применяется для выделения различных структур, таких как кости или опухоли, если они отличаются по яркости от окружающих тканей.

Фильтр Гаусса, также называемый гауссовым сглаживанием, использует гауссово распределение для размытия изображения. Этот метод хорошо подходит для удаления гауссового шума, сохраняя при этом плавные переходы.

Медианный фильтр — это нелинейный фильтр, который заменяет значение пикселя на медиану значений в окрестности пикселя. Он особенно эффективен для подавления импульсного шума («соль и перец»), так как удаляет пиксели с экстремальными значениями, не размывая края.
